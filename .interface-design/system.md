Direction and feel:
- Operational logistics control surface. Calm, dense, and readable.
- Visual language from fleet ops: graphite/metal neutrals + safety yellow accent + semantic status colors.
- Navigation should feel like a control panel: clear counts, stable hierarchy, minimal noise.

Depth strategy:
- Subtle shadows + soft borders. Cards and content areas use light shadows with low-contrast borders.

Spacing base unit:
- 8px grid. Use 8/12/16/24 for spacing and padding.

Key component patterns:
- Navigation: vertical tabs with counts, active state uses accent background and inset ring.
- Cards/Sections: section blocks with soft border, rounded 14-16px, slight shadow.
- Forms: field groups with section titles, boxed inputs (not underlined), focus ring uses accent.
- Tables: soft header separation, row hover tint, semantic status chips.
- Status chips: pill badges with semantic colors (ok/warn/danger/info/muted).
- Inline select: compact with consistent padding and border.
